## 第 1 周：基础夯实 + RAG 强化

### 周一 — Python 基础速通

- **上午**：过一遍 Python 核心语法（列表推导式、装饰器、上下文管理器、异常处理）—— 对照你的 TS 知识快速扫盲
- **下午**：用 Python 重写你原来 Express 项目里的一个小工具函数（比如意图路由），强制自己不用 AI 辅助
- **晚上**：笔记《Python 中与 JS/TS 的核心差异》+ 面试题「Python 中如何实现异步？」

### 周二 — LLM 基础直觉

- **上午**：吴恩达 Prompt Engineering 课程 1-3 节，记笔记
- **下午**：在你项目里做 3 个实验：
  1. Temperature 0.0 / 0.7 / 1.5 对比
  2. 同一个 prompt 调 5 次观察多样性
  3. System Prompt 有无的差异
- **晚上**：整理 3 道面试题回答（Temperature、System Prompt、Token 计算）

### 周三 — Embedding 与向量数据库深入

- **上午**：读 bge-m3 论文摘要 + MTEB 榜单，对比 text-embedding-3-small / ada-002
- **下午**：在你的项目里加一个脚本，批量计算 50 条数据的 embedding，用余弦相似度做检索准确率统计
- **晚上**：笔记《Embedding 模型选型决策树》+ 面试题「为什么选 bge-m3？」

### 周四 — 混合检索实现

- **上午**：学习 BM25 原理（看 Elasticsearch 文档的 BM25 章节 30 分钟）
- **下午**：在你的 Python 后端里集成 BM25（用 rank_bm25 库），实现向量 + BM25 混合检索，加权融合结果
- **晚上**：笔记《混合检索的权重调优实验》+ 面试题「混合检索的定义及核心作用」

### 周五 — RAG 效果评估体系搭建

- **上午**：学习 RAGAS 框架（看官方文档 quickstart）
- **下午**：准备 50 条测试 Q&A，用 RAGAS 评估你的 RAG 管道（忠实度、相关性、召回率），记录基线分数
- **晚上**：笔记《我的 RAG 系统评估基线报告》+ 面试题「RAG 效果评估的方法」

### 周六 — 文档处理与分块策略实验

- **上午**：学习 PDF 解析工具对比（PyPDF2、pdfplumber、MinerU）
- **下午**：在你项目里测试不同 chunk_size（256 / 512 / 1024）和 overlap（0 / 50 / 100）对检索命中率的影响
- **晚上**：笔记《分块策略实验数据》+ 面试题「分块策略及差异」

### 周日 — 休息 + 轻量复盘

- 回顾本周所有笔记，更新 README 进度
- 在脑海中过一遍 RAG 全链路（洗澡时自己讲一遍）
- 可选：看一篇 RAG 最新论文的摘要（如 Self-RAG、GraphRAG）

## 第 2 周：Agent 开发 + 工程化

### 周一 — Function Calling 深入

- **上午**：读 DeepSeek Function Calling 文档 + OpenAI Function Calling 最佳实践
- **下午**：在 baby-ai 里实现第一个工具「天气查询」：用户问"今天适合带宝宝出门吗？"→ Agent 调用天气 API → 结合 RAG 生成建议
- **晚上**：笔记《Function Calling 完整链路》+ 面试题「Tool Calling 的完整链路」

### 周二 — ReAct 模式实现

- **上午**：读 ReAct 论文摘要 + LangChain ReAct 源码片段
- **下午**：给你的 Agent 加上 ReAct 循环：思考 → 行动 → 观察 → 再思考，用日志打印每一步
- **晚上**：笔记《ReAct 模式实践》+ 面试题「ReAct 的工作原理」

### 周三 — MCP 协议与接入

- **上午**：读 MCP 官方文档 + 看一个 MCP Server 示例（如 fetch 或 filesystem）
- **下午**：给 baby-ai 接入一个 MCP Server（如联网搜索），让 Agent 能搜最新信息
- **晚上**：笔记《MCP 接入实战》+ 面试题「MCP 与 Function Calling 的差异」

### 周四 — Agent 记忆系统设计

- **上午**：学习 Agent 记忆类型（短期/长期/工作记忆）及存储方案
- **下午**：在 baby-ai 里实现「跨会话记忆」：提取用户宝宝月龄、过敏史，存为长期记忆，下次对话自动注入
- **晚上**：笔记《Agent 记忆系统设计》+ 面试题「Agent 的 Context Window 约束」

### 周五 — Docker 与测试

- **上午**：学 Docker 多阶段构建，写一个生产级的 Dockerfile
- **下午**：用 pytest + FastAPI TestClient 写 5 个接口测试（正常/异常/边界）
- **晚上**：笔记《Docker 化与自动化测试》+ 面试题「生产环境中使用 Docker 的注意事项」

### 周六 — 流式输出生产优化

- **上午**：学习 SSE 规范深入（断点续传、错误重试、心跳机制）
- **下午**：在你的流式接口里加入：
  1. 中断处理（用户停止时释放资源）
  2. 错误兜底（LLM 报错时返回已生成内容）
- **晚上**：笔记《流式输出生产级优化》+ 面试题「流式返回中网络中断的处理方法」

### 周日 — 休息 + 周复盘

## 第 3 周：生产落地 + 微调概念

### 周一 — 模型部署方案对比

- **上午**：学 vLLM 原理（PagedAttention、Continuous Batching），对比 Ollama
- **下午**：用 vLLM 部署一个开源模型（如 Qwen2-1.5B），测吞吐量
- **晚上**：笔记《Ollama vs vLLM 选型》+ 面试题「Ollama 与 vLLM 的差异」

### 周二 — 安全防护

- **上午**：学 OWASP LLM Security Top 10，重点 Prompt Injection
- **下午**：给你的 /api/chat 加上：
  1. 输入敏感词过滤
  2. 防御性 System Prompt
  3. 输出内容审核
- **晚上**：笔记《AI 应用安全防护实践》+ 面试题「Prompt Injection 的防御方法」

### 周三 — Token 成本控制

- **上午**：学 Token 成本计算模型，对比不同模型的定价
- **下午**：在你的项目里加入：
  1. 每次请求的 Token 用量统计
  2. 缓存高频问答（Redis 或内存缓存）
  3. 成本估算接口
- **晚上**：笔记《Token 成本优化方案》+ 面试题「Token 消耗与模型成本的控制方案」

### 周四 — 微调原理

- **上午**：看吴恩达 Finetuning 课程前 2 节，理解 SFT vs RLHF
- **下午**：用 unsloth 跑一个 LoRA 微调 demo（数据集用你的育儿 Q&A）
- **晚上**：笔记《LoRA 微调初体验》+ 面试题「LoRA 的原理」

### 周五 — 微调与 RAG 对比

- **上午**：学习 RAG vs Fine-tuning 决策框架
- **下午**：用同一批测试数据，对比微调后模型 vs RAG 增强的效果（准确率、幻觉率、成本）
- **晚上**：笔记《RAG vs Fine-tuning 选型决策树》+ 面试题「RAG 与 Fine-tuning 的选型方法」

### 周六 — 项目集成与文档

- **上午**：写 baby-ai 的完整技术文档（架构图、接口文档、部署指南）
- **下午**：录制一个 5 分钟的项目演示视频（屏幕录制 + 讲解）
- **晚上**：更新 README，整理 GitHub 仓库

## 每日必须坚持的 3 个习惯

1. **早晨 5 分钟**写下当天 3 个具体目标（如：实现混合检索、写 2 道面试题回答、跑通 vLLM demo）

3. **每完成一个实验，立刻记笔记**（别等"以后有空再整理"）
